## Quick Reference

### By Category

- **Continuity Glyphs** - Session persistence and memory linkage
- **Identity Glyphs** - Agent personas and signatures
- **State Glyphs** - System status indicators
- **Event Glyphs** - Milestones and transitions
- **Reflection Glyphs** - Self-reference and mirroring
- **Governance Glyphs** - Trust and compliance
- **Navigation Glyphs** - Movement and direction
- **Data Glyphs** - Information and storage

See [Glyph Catalog](glyph_catalog.md) for complete details.

### By Use Case

- **Glyphtrail** → continuity.*, events.*, navigation.forward
- **LingOS** → reflection.*, continuity.infinity, state.*
- **Active MirrorOS** → identity.*, state.*, governance.*
- **MirrorDNA Protocol** → identity.dna, continuity.*, governance.*
